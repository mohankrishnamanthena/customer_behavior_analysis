# customer_behavior_analysis
# 📊 Customer Behavior Analysis using Python, SQL & Power BI

## 📌 Project Overview

This project analyzes customer shopping behavior to identify purchasing patterns, customer segments, and sales trends. The workflow includes data cleaning, exploratory data analysis, SQL-based business analysis, and an interactive Power BI dashboard for visualization.

The objective is to demonstrate practical data analytics skills by transforming raw customer data into actionable business insights.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- PostgreSQL
- SQLAlchemy
- SQL
- Power BI
- Jupyter Notebook

## 📈 Project Workflow

1. Import the customer dataset.
2. Clean and preprocess the data using Python.
3. Handle missing values and prepare the dataset.
4. Load the cleaned data into PostgreSQL.
5. Perform business analysis using SQL queries.
6. Build an interactive dashboard in Power BI.
7. Generate insights and business recommendations.

## 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/yourusername/customer-behavior-analysis.git
```

### Install dependencies

```bash
pip install pandas numpy sqlalchemy psycopg2-binary
```

### Run the notebook

Open the Jupyter Notebook and execute the cells in order.

### Load data into PostgreSQL

Update the database credentials in the notebook and execute the SQL upload section.

### Open Power BI

Open the `.pbix` file and refresh the data source.
